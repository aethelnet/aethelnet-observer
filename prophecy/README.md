# PROPHECY Alert Center - "The Oracle's Eye"

**Port:** 1428  
**Purpose:** Real-time PROPHECY event feed - Major Move predictions from THOTH oracle

---

## 🎯 What It Does

The PROPHECY Alert Center displays high-confidence predictions of major price movements (>1.5% or resonance >0.9) **before they happen**. This is the system that **generates news** rather than reacting to it.

---

## 🚀 Quick Start

```bash
cd prophecy
./serve.sh
# Open http://localhost:1428
```

---

## 👁️ Features

### Active PROPHECY Events Feed
- Real-time list of active PROPHECY events
- Auto-refreshes every 5 seconds
- Shows only "live" PROPHECY events (not expired)

### PROPHECY Card Display
Each card shows:
- **Large predicted move %** (e.g., "+3.2%")
- **Countdown timer** (e.g., "In 8 minutes")
- **Resonance meter** (visual confidence indicator, 0-1 scale)
- **Symbol & Direction** (BUY/SELL with color coding)
- **Intensity indicator** (MAX INTENSITY badge for intensity >= 2.0)
- **Current & Target Price**
- **Quick action button** - "View Chart" → Opens chartview

### Visual Hierarchy
- **Gold/Red urgency** - High resonance or imminent moves
- **Thicker borders** - Higher resonance = more prominent
- **Pulsing animation** - For events happening soon (<10 minutes)

### Summary Stats
- Total active PROPHECY events
- High resonance count (>0.9)
- Average predicted move %
- Events expiring soon (<10 min)

### Filtering & Sorting
- **Sort by:** Resonance, Predicted Move, Time Remaining, Intensity
- **Filter by Type:** BUY only, SELL only, All
- **Min Resonance:** Slider (0-1)
- **Symbol Filter:** Filter by specific trading pair

---

## 📊 Data Source

**Primary:** `/api/dashboard/signals` - Filtered for `is_prophecy: true`

**Data Fields:**
- `symbol` - Trading pair
- `predicted_move_percent` - Expected price movement percentage
- `resonance` - Confidence level (0-1, where 1.0 = 100%)
- `intensity` - Signal intensity (1.0 or 2.0 for MAX INTENSITY)
- `direction` - BUY or SELL
- `prophecy_type` - Type of PROPHECY (usually "MAJOR_MOVE")
- `timestamp` - When PROPHECY was detected
- `price` - Price at detection time

## 🔮 What is PROPHECY?

PROPHECY events are generated by the **THOTH oracle** (the ML prediction engine) when it detects:
- **Major moves** (>1.5% predicted price change), OR
- **High resonance** (>0.9 confidence)

These are high-confidence predictions of significant price movements that are about to happen. The system doesn't wait for news - it **generates** the prediction.

---

## 📝 Notes

- PROPHECY events are time-sensitive - they expire after the estimated time horizon
- Higher resonance = higher confidence in the prediction
- MAX INTENSITY events (intensity >= 2.0) are the most confident predictions
- The system automatically filters out expired PROPHECY events
