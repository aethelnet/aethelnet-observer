## 🎮 How to Use

1. **View Network** - See 3D network of market connections
2. **Click Node** - Click a market in the sidebar to select it
3. **View Chart** - Price chart appears at bottom when node is selected
4. **Filter** - Use slider to filter connections by correlation strength
5. **Interact** - Drag to rotate, scroll to zoom
