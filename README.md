# Aethelnet Observer (Legacy Interface)

> ⚠️ **NOTICE: ARCHITECTURE SHIFT**
> 
> This repository contains the historical grid-based MVP frontend for the Aethelnet project.
>
> We have radically shifted our philosophy! The entire system has evolved into **LGNN (Liquid Graph Neural Networks)**, a continuous-time, node-based graph architecture.
> 
> **The active workspace, engine, and frontend are now unified in the [auratic-systems-prime](https://github.com/ProphitEngine/ProphitEngine) repository.**

## 📜 Historical Context

This repository is preserved as an open-source reference for:
- Historical WebSocket integration patterns.
- High-performance, read-only data binding via Pinia and Vue 3.
- Native `Canvas` / WebGL charting experiments.

If you are looking to run, develop, or interact with the actual **Sovereign Engine** or the **Glassmorphism Node Canvas**, please head over to the main `auratic-systems-prime` repository!

## Galaxy Blueprint View
The Observer now includes the massive **CodeSpider Blueprint Engine**.
You can open the Command Palette (`Cmd+K` or `/`) and type **"Load System Blueprint"** to instantly render the backend node's 181,000+ file/function dependencies as a 3D WebGL Galaxy using Three.js!
