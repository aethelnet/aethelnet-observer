# Aethelnet Observer

The **Aethelnet Observer** is the visual frontend (Dashboard) for the continuous-time Liquid Graph Neural Network (LGNN) nodes.

It connects to your local `aethelnet-node` daemon and visualizes the chaotic, organic topology of the network in real-time.

## Setup
Currently, this is a vanilla HTML/JS application. Simply serve the directory with a web server or open `index.html`.
